## 重点领域标准索引

### 1. 数据安全与个人信息保护

> 随着“数安法”和“个保法”的落地，配套标准已形成体系。

* **分类分级**：GB/T 43697-2024《数据安全技术 数据分类分级规则》（**核心**）
* **风险评估**：GB/T 45577-2025《数据安全技术 数据安全风险评估方法》（待实施）
* **APP合规**：
* GB/T 43739-2024《数据安全技术 应用商店的移动互联网应用程序（App）个人信息处理规范性审核与管理指南》
* GB/T 41391-2022《信息安全技术 移动互联网应用程序（App）收集个人信息基本要求》


* **去标识化**：GB/T 42460-2023《信息安全技术 个人信息去标识化效果评估指南》
* **出境安全**：TC260-PG-20222A《网络安全标准实践指南—个人信息跨境处理活动安全认证规范》

### 2. 人工智能与算法安全 (New!)

> 应对大模型、深度伪造等新兴技术的监管要求。

* **生成式AI**：TC260-003《生成式人工智能服务内容标识方法》（征求意见稿）
* **标注安全**：TC260-PG-2024《网络安全标准实践指南——生成式人工智能数据标注安全规范》
* **人脸识别**：GB/T 41819-2022《信息安全技术 人脸识别数据安全要求》
* **机器学习**：GB/T 42888-2023《信息安全技术 机器学习算法安全评估规范》

### 3. 网络安全等级保护 (MLPS 2.0)

> 等保2.0依然是国内合规的基础线。

* **基本要求**：GB/T 22239-2019《信息安全技术 网络安全等级保护基本要求》
* **定级指南**：GB/T 22240-2020《信息安全技术 网络安全等级保护定级指南》
* **测评要求**：GB/T 28448-2019《信息安全技术 网络安全等级保护测评要求》
* **云计算扩展**：GB/T 31168-2023《信息安全技术 云计算服务安全能力要求》

### 4. 关键信息基础设施 (CII)

* **保护要求**：GB/T 39204-2022《信息安全技术 关键信息基础设施安全保护要求》
* **边界识别**：已发布各行业（金融、电信、能源等）CII 认定规则（通常不公开发布，需咨询行业主管单位）。
